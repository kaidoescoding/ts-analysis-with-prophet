Chapter 1: Decomposing the Time Series

Remember when I wanted you to ask yourself why we should even concern ourselves about time series? Well a time series deals with data in a unique way. Time series are essentially some sort of combination of three different parameters - trend, seasonality and irregular noise. This makes it exceptionally easy to forecast and you will see that Prophet will make this task even easier! R can allow us to view these parameters directly.

decomp <- decompose(ENSO)
decomp$x # returns the raw time series data

# We can view the three aforementioned components.

decomp$trend # trend
decomp$seasonal # seasonality
decomp$random # noise
# We can also gain insight the following: 

decomp$type # nature of the object - are the three aforementioned parameters linked in an additive model or a multiplicative model?
## [1] "additive"
decomp$figure # tells us the average seasonal effect for each increment in the cycle. In this case, we have a vector of twelve elements, which makes sense because there are twelve months in a year.

Chapter 2: Linear Regression

lm(ENSO ~ time(ENSO)) # R uses least squares regression to estimate a line of best fit.
## 
## Call:
## lm(formula = ENSO ~ time(ENSO))
## 
## Coefficients:
## (Intercept)   time(ENSO)  
##   -4.867203     0.002532
cf <- coef(lm(ENSO ~ time(ENSO))) # creates a vector of the coefficients of the line of best fit.

invisible(plot(ENSO, main = "ENSO", ylab = "Temperature Change") + abline(a = cf[1], b = cf[2], lwd = 2, col = "red"))

From this plot, we can see that the change in temperature is pretty much constant from 1951 to 2022. This means that we can make the following inference: the temperature of the waters in the central and eastern tropical Pacific Ocean have stayed the same since 1951.

Chapter 4: Plotting the forecast using ggplot2

We can now plot this; it should be noted that historical data and predictions should clearly be distinguished by some sort of visual difference. In this case, I will put a separation line.

library(ggplot2) # useful tool for complex plots

# Ensure dates are correctly formatted
df$ds <- as.Date(df$ds)
pdfForecast$ds <- as.Date(pdfForecast$ds)

# Group the predicted data together using a Boolean
future_only <- pdfForecast[pdfForecast$ds > max(df$ds), ]

# Craft the plot from the ground up
ggplot() +
  # Historical ENSO data
  geom_line(data = df, aes(x = ds, y = y), color = "black") + # using aes to make the plot visually appealing
  
  # Full forecast (fitted + predicted)
  geom_line(data = pdfForecast, aes(x = ds, y = yhat), color = "blue") +
  
  # Confidence intervals for full forecast
  geom_ribbon(data = pdfForecast, aes(x = ds, ymin = yhat_lower, ymax = yhat_upper),
              alpha = 0.2, fill = "blue") +
  
  # Future-only forecast emphasised
  geom_line(data = future_only, aes(x = ds, y = yhat), color = "green", size = 1) +
  
  # Vertical line separating historical vs forecast
  geom_vline(xintercept = max(df$ds), linetype = "dashed", color = "red") +
  
  # Labels & theme
  labs(title = "ENSO Index: Historical & Short-Term Forecast (5 Years)",
       subtitle = "Black = historical | Blue = fitted+forecast | Green = future-only",
       x = "Date",
       y = "ENSO") +
  theme_minimal() +
  theme(plot.title = element_text(face = "bold", size = 14))

Epilogue

It appears that the overall trend of the temperature series is stationary. There is a slight upward trend but for almost seven decades of opportunity, it isn’t enough to be recognised. The ENSO daily index shows very intense seasonal oscillations around a mean which has been stable for 71 years. Prophet used this stable mean and seasonal oscillation exhibition to forecast values in the next five years which mimic the historical seasonality around this mean. The confidence interval (highlighted region on the plot) provides a level of variability which is expected. To conclude, the time series analysis has shown that most of the variation in ENSO’s index is influenced by predictable yearly fluctuations rather than a long-term incline/decline. The forecast has provided a reliable extension of these patterns to serve in the interests of analysts.
